# Sykkeldagbok

Privat dagbok for sykling, mat, restitusjon og følelse. Målet er å samle daglige notater i et strukturert skjema, slik at det blir enklere å se mønstre over tid.

## Første idé

Programmet skal brukes etter hver dag eller økt. Det skal fungere godt på telefon, men også kunne brukes på PC. Første versjon bør være enkel, rask og trygg å fylle ut.

## Hovedmål

- Registrere om dagen inneholder trening, ritt, hvile eller annet.
- Skrive hva som ble spist, omtrent hvor mye og når.
- Logge sykkeløkter med varighet, intensitet og følelse.
- Notere søvn, energi, stress, restitusjon og kroppsfølelse.
- Se mønstre over tid, for eksempel mellom mat, belastning, søvn og prestasjon.
- Kunne dele resultater senere, for eksempel som eksport, rapport eller skjermbilde.

## Foreslått første løsning

En liten mobilvennlig web-app:

- Kan åpnes i nettleser på telefon.
- Skjemaet lagrer data lokalt først.
- Senere kan vi legge til eksport til CSV, Excel eller PDF.
- Senere kan vi legge til innlogging/synk hvis du vil bruke den på flere enheter.
